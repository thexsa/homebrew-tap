# 🍺 Homebrew Tap

Custom [Homebrew](https://brew.sh) formulas.

## Available Formulas

| Formula | Description |
|---------|-------------|
| [peep](https://github.com/thexsa/peep) | TLS diagnostic tool — peeps into handshakes and certificate chains |

## Installation

```bash
brew install thexsa/tap/peep
```

Or add the tap first:
```bash
brew tap thexsa/tap
brew install peep
```

## Updating

```bash
brew update
brew upgrade peep
```
